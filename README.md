## Dependencies

Please ensure that you have:

- `python` installed, ideally through `pyenv`
- `poetry` installed

## Development

For first-time setup, run

```bash
python3 -m .venv
poetry install
pre-commit install --hook-type pre-commit --hook-type pre-push
```

Then, to get started, run

```bash
source .venv/bin/activate
```
