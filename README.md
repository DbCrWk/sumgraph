## Authors

- Zara Memon

## Dependencies

Please ensure that you have:

- `python` installed, ideally through `pyenv`
- `poetry` installed
- `make` installed

## Development

For first-time setup, run

```bash
make setup
```

Afterwards, when working on the project, run

```bash
source .venv/bin/activate
```
